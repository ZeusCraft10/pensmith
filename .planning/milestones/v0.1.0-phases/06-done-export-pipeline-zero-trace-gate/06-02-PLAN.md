<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| draft text → DDG query string | Untrusted draft prose becomes a search query; must be URL-encoded, host hard-coded (no arbitrary fetch / SSRF) |
| DDG HTML response → parser | Untrusted remote HTML; parsed as plain text/regex only (no DOM eval, no innerHTML) |
| plagiarism.ts → network | Must route exclusively through http.ts (ESLint chokepoint) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-02-01 | Tampering / SSRF | DDG query URL construction | mitigate | Host hard-coded to `https://html.duckduckgo.com/html/`; only the `q` param is dynamic and is `encodeURIComponent`-escaped; source 'generic' bucket enforced |
| T-06-02-02 | Tampering | DDG HTML response parsing | mitigate | parseDdgHtml uses regex/String only — no eval, no innerHTML, no DOM (V5 input validation); malformed HTML → empty matches, never a crash |
| T-06-02-03 | Denial of Service | DDG rate-limit / block on burst | mitigate | maxPhrases cap (10/paper) + Semaphore(5) + http.ts generic TokenBucket (5 RPS) + browser-like headers (06-RESEARCH Pitfall 5); errors swallowed advisory |
| T-06-02-04 | Elevation of Privilege | plagiarism hit escalating to a hard block | mitigate | Advisory by construction — returns an array, no blocking verdict; only DONE-09 gate (06-05) can pause on a hit, and only with user confirm |
| T-06-02-SC | Tampering | npm/pip/cargo installs | mitigate | No package installs in this plan (jszip added in 06-01); nothing to slopcheck here |
