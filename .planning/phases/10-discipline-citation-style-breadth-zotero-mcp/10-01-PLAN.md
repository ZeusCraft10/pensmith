| T-10-01-02 | Denial of Service | template re-registration | mitigate | registeredStyles Map guards every templates.add so a hot loop of renderStyle calls registers each style exactly once (Pitfall 1); proven by the back-to-back determinism test. |

| T-10-01-04 | Information Disclosure | render-time network | mitigate | Render is OFFLINE — readFileSync of committed .csl + bundled en-US citeproc locale, no fetch. Determinism test indirectly proves no network variance. |
| T-10-01-SC | Tampering | npm/pip/cargo installs | mitigate | NO installs (citation-js bundles plugin-csl). RESEARCH Package Legitimacy Audit [OK] for all transitive deps. |
