| advisory Pass 2/4 + plagiarism issues → export decision | The gate must surface issues and require confirm; it must NOT auto-block (VRFY-07) yet must NOT silently export over issues (Core Value) — only --yolo bypasses |

| produced export artifact → user | Must carry zero pensmith trace (verb-level surface of DONE-07) |
