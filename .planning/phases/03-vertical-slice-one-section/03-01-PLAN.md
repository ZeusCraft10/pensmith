    Implementation requirements:

    1. **jaroWinkler:**
       - Pre-normalize both inputs: `const A = nfkcNormalize(a).toLowerCase(); const B = nfkcNormalize(b).toLowerCase();`
       - If `A === B` return 1.
       - If either empty, return 0 (except both empty → 1).
       - Compute classic Jaro distance: matching window = `Math.floor(Math.max(A.length, B.length) / 2) - 1`, count matches + transpositions, jaro = (m/|A| + m/|B| + (m-t)/m) / 3.
       - Apply Winkler boost: scan common prefix up to length 4, jw = jaro + prefix * 0.1 * (1 - jaro).
       - Return jw clamped to [0, 1].

    2. **levenshtein (internal):** classic 2-row DP, O(|a|·|b|) time, O(min) space.

    3. **levenshteinSubstring:**

       - If `needle.length === 0` return 1 (or 0 — pick the version tested in fuzzy.test.ts).
       - If `needle.length > haystack.length` return 0.
       - Slide a window of `needle.length` chars across haystack, compute levenshtein(window, needle), track min.
       - Return `1 - (minDist / needle.length)`.
       - Optimization (optional, only if needed for the 200-run property tests to finish <5s): use Ukkonen's banded DP. NOT required for correctness.
